= Customizing VMs
:context: virt-customizing-vms
:virt-customizing-vms:

toc::[]

You can customize a virtual machine (VM) as you create it. You can also create a customized template to use for additional new VMs.

You can also use the web console to create a VM xref:../../../virt/creating_vm/virt-creating-vms-from-instance-types.adoc#virt-creating-vm-instancetype_virt-creating-vms-from-instance-types[with predefined instance type settings].
