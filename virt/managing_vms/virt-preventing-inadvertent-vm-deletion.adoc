= Controlling virtual machine states
include::_attributes/common-attributes.adoc[]
:context: virt-preventing-inadvertent-vm-deletion

toc::[]

You can prevent the inadvertent deletion of a virtual machine (VM) by enabling delete protection from either the {product-title} web console or the command line.

You can also disable delete protection for a VM from the web console or the command line.

.Prerequisites

* You have installed {VirtProductName} 4.19 or later.
* You have installed the OpenShift CLI (`oc`).
