:context: virt-about-CCLM-live-migration

toc::[]

Cross-cluster live migration enables users to move a virtual machine (VM) workload from one {product-title} cluster to another {product-title} cluster without disruption.

:FeatureName: Cross-cluster live migration
include::snippets/technology-preview.adoc[]

.Prerequisites

* {VirtProductName} 4.20 or later must be installed.

* The pre-release version of the {mtv-first:} (MTV) must be installed.

* The {product-title} and {VirtProductName} minor release versions must match (e.g., 4.20.z).

* You must have cluster administration permissions and appropriate RBAC privileges to manage VMs on both clusters.

* The clusters must be connected to the same L2 network segment.
